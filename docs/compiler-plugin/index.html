With compiler plugins the compiler can be extended.
The compiler defines extension points, an extension point provides an interface which can be implemented by the plugin. These extensions are registered in the plugin.xml of the plugin. 
<?xml version="1.0" encoding="UTF-8" ?>
<plugin>
    <id>io.github.potjerodekool.nabu.plugin.simple</id>
    <version>0.0.1</version>
    <description>Simple plugin for Nabu compiler</description>
    <extensions>
    </extensions>
</plugin>
