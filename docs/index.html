Nabu compiler documentation

Introduction

The Nabu compiler is a polyglot compiler for compiling source code into Java bytecode so it can be executed on the Java Virtual Machine (JVM).
Currently, the Nabu compiler only supports the Nabu language but in the future it will be possible to support for other programming languages.
Interoperability with other languages is possible with a language parser, currently there is a language parser for Java so you can use classes, invoke methods and such.
